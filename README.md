Esse é um app Android bem simples que fiz pra praticar navegação entre telas usando Jetpack Compose. A ideia é básica: tem uma tela de login onde você clica em "Entrar" e vai pro menu principal. No menu tem três opções, Perfil, Pedidos e Sair. Cada tela dessas tem só um título e um botão de voltar, nada muito elaborado. O foco mesmo foi aprender a usar o Navigation Compose pra transitar entre as telas passando o NavController. Cada tela tem uma cor diferente pra diferenciar visualmente (rosa, azul, verde, etc). É um projetinho de estudo, direto ao ponto, sem banco de dados nem nada complexo, só navegação pura entre 4 telas.
